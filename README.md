# PLPBasicGitAssignment

Add hello.txt with a greeting

STEPS AND COMMAND USED :

-first log in to GitHub and create a new repository named "PLPBasicGitAssignment,"

-initializing it with a README file. 

-On the local machine, create a folder named "PLPBasicGitAssignment," 

-open a terminal, navigate to the folder, and initialize a Git repository using git init. 

-Link the local repository to the GitHub repository with git remote add origin <repository-url>, 

-replacing <repository-url> with the GitHub repository URL. Create a file named hello.txt with the content "Hello, Git!", 

-stage the file with git add hello.txt, and commit the changes with git commit -m "Add hello.txt with a greeting". 

-Push the changes to GitHub using git push -u origin main, then verify that the hello.txt file and commit message are visible in the GitHub repository.
